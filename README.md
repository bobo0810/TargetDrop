## 环境

| python版本 | pytorch版本 | 系统   |
|------------|-------------|--------|
| 3.6        | 1.6.0       | Ubuntu |

## 使用

```python
import TargetDrop
class MyNet():
    def __init__(self):
        ...
        self.target_drop=TargetDrop(in_channels)
    def forward(self, x):
        ...
        x=self.target_drop(x)
```
